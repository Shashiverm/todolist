## Features

- Create, mark, and delete tasks
- Mark tasks as complete/incomplete
- User-friendly interface
- Local data storage


## Usage

1. Open the web app
2. Fill in task details
3. Click the add button to add task
4. Tap on tasks to mark them complete
5. Delete tasks
